# Магнат 🎩🎲

Уютная настолка про недвижимость в духе «Монополии» (без копирайтов), сделанная как Telegram Mini App. Русский интерфейс, тёплый премиальный дизайн, игра с ботами и с друзьями по коду.

Скупай улицы, вокзалы и предприятия, собирай цветные группы в монополии, застраивай их домами и отелями и разоряй соперников. Последний платёжеспособный магнат за столом побеждает.

## Что внутри

- **Соло против ботов** — три уровня сложности (лёгкий, средний, сложный).
- **Быстрая игра** — публичный подбор соперников; стол мгновенно наполняется, а свободные места незаметно занимают боты.
- **Игра с друзьями** — общая комната по коду из четырёх символов, пустые места добираются ботами при старте.
- Детерминированный игровой движок (общий для клиента и сервера), покрытый тестами.
- Классическое квадратное поле из 28 клеток, кубики, аренда, монополии, дома и отели, тюрьма, «Шанс» и «Казна», налоги, банкротство.

## Стек

- **app** — React + Vite + Zustand, SVG-поле, синтезированные звуки (WebAudio), Telegram WebApp SDK.
- **server** — Hono + grammY (бот) + better-sqlite3, авторитетные онлайн-комнаты в памяти.
- **shared** — движок, боты, поле, колоды, проекция вида. Один код на клиенте и сервере.

## Локальный запуск

```bash
npm install
npm run dev:server   # сервер на :3000 (DEV-режим без BOT_TOKEN)
npm run dev:app      # клиент на :5174 с прокси на /api
```

Соло-игра работает полностью на клиенте и не требует сервера.

## Проверки

```bash
npm test        # тесты движка (поле, аренда, монополии, тюрьма, банкротство, полные партии ботов)
npm run typecheck
npm run build
```

## Деплой (Railway)

1. Задай переменные из `.env.example` (как минимум `BOT_TOKEN`, `APP_URL`, `JWT_SECRET`, `BOT_USERNAME`).
2. `npm install && npm run build` (см. `railway.json`).
3. `npm start` поднимает сервер, отдаёт собранный клиент и настраивает вебхук Telegram.

Node зафиксирован на 22.x (`.nvmrc` + `engines`), чтобы у `better-sqlite3` ставился готовый бинарник.
